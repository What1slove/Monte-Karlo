Вычисление определённых интегралов методами Монте-Карло

Правила описания формул

Операторы

  • + -Сложение
  • - -Вычитание
  • * -Умножение
  • / -Деление
  • % -Остаток от деления
  • ^ -Возведение в степеь

Унарные операторы

  • x! -Факториал
  • abs x -Модуль
  • sqrt x -Корень
  • sin x, cos x, tan x -Тригометрические функции
  • asin x, acos x, atan x -Обратные тригометрические функции
  • sinh x,asinh x.... -Гипорболические аналоги предидущих 2-х пунктов
  • ln x, log10 x -Натуральный и десятичные логарифмы
  • exp x -Экспонента
  • floor x, round x, ceil x, trunc x -Функции округления до целого

Предопределённые функции

  • random(n) -случайное число от 0 до n
  • min(a,b,…),max(a,b,…) -Минимальное/Максимальное число из набора
  • pow(x, y) -Возведение в степень
  • atan2(y, x) -Угол между (0,0) и (x,y) в радианах
  • roundTo(x, n) -Округляет x до n цифр после запятой

В описании функции может быть только одна переменная, x
